# 中国油价地图网站 - 任务规划

## 📋 项目概述

**目标：** 单屏展示中国各省实时油价的地图可视化网站

**核心功能：**
1. 中国地图（各省着色）
2. 鼠标悬浮显示近 5 次调价信息
3. 每日自动更新数据

---

## 🎯 技术栈

| 组件 | 技术选型 |
|------|----------|
| 前端框架 | HTML + CSS + JavaScript (纯静态) |
| 地图库 | ECharts (百度开源，支持中国地图) |
| 数据源 | 手动更新/爬虫 + JSON |
| 部署 | 静态托管 (Vercel/Netlify/GitHub Pages) |
| 自动更新 | GitHub Actions + 定时任务 |

## 🔍 数据来源

### 油价数据
- **官方来源：** 国家发改委价格司
- **聚合网站：** 
  - 金投网 (https://www.cngold.org/)
  - 油价网 (http://www.youjia100.cn/)
  - 易车网油价频道

### 数据格式
```json
{
  "province": "广东省",
  "provinceCode": "440000",
  "prices": {
    "92": 7.85,
    "95": 8.45,
    "98": 9.25,
    "0": 7.15
  },
  "updateTime": "2026-03-19"
}
```

## 🎨 设计方案

### 颜色映射
- 低油价 → 绿色 (#91CC75)
- 中油价 → 黄色 (#FAC858)
- 高油价 → 红色 (#EE6666)

### 悬浮提示内容
```
广东省
────────────
92# ¥7.85 | 95# ¥8.45

2026-02-15 ↑ +0.15
2026-02-01 ↑ +0.10
2026-01-18 ↓ -0.20
```

## ⚠️ 风险点

1. **数据源稳定性** - 需要可靠的油价 API 或爬虫
2. **地图边界问题** - 使用标准 GeoJSON 避免争议
3. **更新频率** - 国内油价每 10 个工作日调整一次
